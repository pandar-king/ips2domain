# ips2domain - IP反查域名工具

IP反查域名工具，支持批量查询IP关联域名、提取主域名、查询ICP备案信息，并导出Excel报告。

## 功能特性

- **IP反查域名**：通过IP地址查询关联域名
- **智能输入清洗**：自动处理URL、IP等多种输入格式
- **主域名提取**：从关联域名中提取主域名
- **ICP备案查询**：自动查询主域名的备案信息
- **Excel导出**：结果导出为Excel，支持高亮显示
- **代理支持**：支持HTTP/HTTPS/SOCKS5代理
- **批量处理**：支持批量查询，每50个提醒更换IP
- **中断恢复**：支持Ctrl+C中断后恢复继续
- **失败重试**：查询失败自动重试3次
- **进度显示**：实时显示查询进度

## 安装依赖

```bash
pip install requests openpyxl
```

## 使用方法

### 基本用法

```bash
# 单个IP查询
python ips2domain.py -u 116.153.40.166

# 批量查询（从文件读取IP列表）
python ips2domain.py -l ipv4.txt

# 指定输出文件
python ips2domain.py -l ipv4.txt -o result.xlsx
```

### 使用代理

```bash
# HTTP代理
python ips2domain.py -l ipv4.txt -p http://127.0.0.1:7890

# SOCKS5代理（推荐FlClash/V2RayN）
python ips2domain.py -l ipv4.txt -p socks5://127.0.0.1:7890

# 简写（自动添加http://）
python ips2domain.py -l ipv4.txt -p 127.0.0.1:7890
```

### 命令行参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `-u, --url` | 单个目标URL或IP | `-u 116.153.40.166` |
| `-l, --list` | 包含目标列表的文件 | `-l ipv4.txt` |
| `-o, --output` | 输出文件名（默认：results.xlsx） | `-o result.xlsx` |
| `-p, --proxy` | 代理地址 | `-p http://127.0.0.1:7890` |
| `-h, --help` | 显示帮助信息 | `-h` |

## 输入文件格式

`ipv4.txt` 格式（每行一个IP或URL）：
```
116.153.40.166
106.225.240.28
http://example.com
https://192.168.1.1
```

## 输出说明

Excel文件包含以下列：

| 列名 | 说明 |
|------|------|
| 原始输入 | 输入的IP或URL |
| IP地址 | 解析后的IP地址 |
| 关联域名 | 查询到的关联域名列表 |
| 主域名及备案信息 | 主域名 + ICP备案号 + 主办单位 |

## 中断恢复

查询过程中按 `Ctrl+C` 中断，进度会自动保存到 `.revip_state.json`。

下次运行时会提示：
```
📂 发现未完成的进度:
   已处理: 25/100
   输出文件: results.xlsx
是否继续上次的进度? (yes/no): yes
```

输入 `yes` 即可继续上次的进度。

## 注意事项

1. **查询延迟**：每个IP查询间隔2-4秒，避免请求过快
2. **更换IP**：每处理50个IP会提醒更换代理IP
3. **ICP查询**：每个主域名查询间隔0.5秒
4. **失败重试**：查询失败会自动重试3次
